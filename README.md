# Picksy 🎯

**두 선택지를 비교해서 내 취향에 더 맞는 쪽을 찾아주는 AI 의사결정 도우미**

🔗 **https://picksy-kohl.vercel.app/**

---

## 어떻게 쓰나요?

1. 고민 중인 두 선택지를 입력 (예: 물 vs 탄산음료)
2. AI가 이 둘을 비교하는 기준 5개를 자동으로 생성
3. 각 기준마다 슬라이더로 내 취향을 표시하고, 별점으로 중요도 설정
4. 결과 확인 — 어느 쪽이 내 취향에 더 가까운지 매칭률로 표시

---

## 주요 기능

- **AI 요소 생성** — 선택지에 맞는 비교 기준을 자동으로 뽑아냄
- **페르소나 선택** — 냉철한 분석가 / 다정한 친구 / 팩트 폭격기 중 선택
- **중요도 가중치** — 별 1~5개로 각 요소의 중요도 조절, 결과에 반영
- **매칭 디테일** — 요소별로 내 취향이 어느 쪽에 가까웠는지 시각화
- **결과 공유** — 결과 텍스트 한 번에 복사해서 공유
